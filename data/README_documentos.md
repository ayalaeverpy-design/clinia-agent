# Documentos de conocimiento de ClinIA

Estos archivos pertenecen a **Clínica Vida Plena**, una institución completamente ficticia creada para el Challenge Alura Agente.

## Archivos

- `politica_privacidad_datos_paciente.pdf`: tratamiento de datos, confidencialidad, conservación y derechos del paciente.
- `politica_cancelaciones_reagendamiento.pdf`: cancelaciones, cambios de fecha, inasistencias, demoras y devoluciones.
- `instrucciones_pre_post_consulta.pdf`: preparación, documentación, teleconsulta, seguimiento y señales de urgencia.
- `preguntas_frecuentes_turnos.csv`: preguntas y respuestas administrativas frecuentes.
- `convenios_coberturas.csv`: planes, copagos, autorizaciones y requisitos ficticios.

## Uso académico

Documento ficticio elaborado exclusivamente para el Challenge Alura Agente. No representa políticas, coberturas ni indicaciones de una institución real.

Los nombres de convenios, montos, correos y condiciones son inventados. El agente no debe utilizar estos documentos para diagnosticar, prescribir medicamentos ni sustituir la atención médica.
